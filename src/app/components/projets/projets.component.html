Une application de growth hacking est une application qui permet d'accroitre la visibilité d'une entreprise sur les reseaux sociaux. Ce projet se concentre sur Twitter et offre la possibilité aux utilisateurs d'automatiser le fonctionnement du compte Twitter en fonction de paramètre prédéfinis. (exemple : retweet des messages contenant le mot "croissance"). Techniquement, ce projet est composé de 3 modules :
- Une application web permettant à l'utilisateur de configurer le comportement du compte.
- Un module communiquant avec l'api de Twitter pour interpreter la configuration et effectuer des actions. (tweet, retweet, favoris...)
- Une API REST servant d'interface entre les autres modules et la base de donnée (MongoDB).
L'orchestration de ces différents modules est effectué à l'aide de Docker.
Actuellement, ce projet n'est pas en production, cependant une demo sera bientôt disponible.
