# Portfolio Angular

Afin d'exécuter le projet pour la phase de développement :

```
git clone https://github.com/Spoutnik87/portfolio.git
```

```
cd portfolio
```

```
npm run dev
```

Afin de construire la version de production :

```
npm run build:prod
```
